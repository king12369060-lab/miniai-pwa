# MiniAI PWA v7：通知與行事曆版

v7 把 MiniAI 從私人助理版升級成更接近日常使用的 PWA。

## 新增功能

| 功能 | 說明 |
|---|---|
| 本機通知 | 可請求瀏覽器通知權限，任務提醒時間到時提示 |
| 任務提醒時間 | 任務可設定 `datetime-local` 提醒 |
| 任務匯出行事曆 | 未完成且有到期日的任務可匯出 `.ics` |
| 到期檢查 | App 開啟與定時檢查今日到期、逾期任務 |
| 每日整理 | 根據任務、聊天、知識整理今日重點 |
| 大型模型後端 | 保留 Vercel `/api/chat` 架構 |
| 本機優先 | 知識、任務、聊天存 IndexedDB |

## 注意

通知是本機輔助通知，不是伺服器推播。

最可靠情境：

- App 開著
- App 被重新開啟
- 手機瀏覽器允許通知

真正背景推播需要 Push API、通知權限、伺服器與推播訂閱管理。

## 行事曆匯出

按：

```text
任務匯出行事曆
```

會產生：

```text
miniai-tasks-YYYY-MM-DD.ics
```

可匯入支援 ICS 的行事曆工具。
