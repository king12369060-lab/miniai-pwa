# MiniAI PWA v2

這是第二版 MiniAI：更適合手機使用的本機優先 PWA。

## 這版升級了什麼？

| 項目 | v1 | v2 |
|---|---|---|
| 知識儲存 | localStorage | IndexedDB |
| 知識管理 | 基礎查看 | 搜尋、排序、刪除 |
| 回答邏輯 | 簡單相似度 | 問題權重 + 內容權重 + 標籤加權 |
| 匯入匯出 | 有 | 有，格式更完整 |
| 手機使用 | 可用 | 更像 App |
| 隱私 | 本機 | 本機優先 |

## 檔案說明

| 檔案 | 用途 |
|---|---|
| `index.html` | 主畫面、知識庫、聊天邏輯 |
| `manifest.json` | PWA 安裝設定 |
| `service-worker.js` | 離線快取 |
| `icon-192.svg` | 小圖示 |
| `icon-512.svg` | 大圖示 |
| `DEPLOY.md` | 手機部署說明 |
| `ARCHITECTURE.md` | 架構說明與下一步升級方向 |

## 怎麼使用？

### 直接測試

可以直接打開 `index.html` 測試基本功能。

注意：如果你是直接從手機檔案管理器打開，部分 PWA 功能可能無法啟用，尤其是 Service Worker 和安裝提示。

### 正式手機安裝

把整個資料夾部署到 HTTPS 網站後，用手機 Chrome 開啟網址，再選擇：

```text
新增到主畫面
```

或瀏覽器跳出「安裝」按鈕時直接安裝。

## 怎麼教它？

你可以按「新增知識」，或直接在聊天框輸入：

```text
學習：問題 => 答案
```

範例：

```text
學習：什麼是 PWA => PWA 是可以像 App 一樣安裝的網頁應用。
```

## 隱私

這版資料存在手機瀏覽器的 IndexedDB。

代表：

- 不需要帳號
- 不會自動上傳
- 清除瀏覽器資料會刪除知識庫
- 換手機不會自動同步
- 建議定期使用「匯出」備份

## 下一步可以升級成什麼？

建議路線：

1. `v2`：本機知識庫 PWA，也就是現在這版
2. `v3`：加入文件匯入，例如 TXT / Markdown / PDF 摘要
3. `v4`：加入後端代理，安全連接大模型
4. `v5`：封裝成 Android APK
