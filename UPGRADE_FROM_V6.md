# 從 v6 升級到 v7

## 建議流程

```text
v6 匯出全部
→ 上傳 v7 檔案
→ Vercel 重新部署
→ 打開 v7
→ 匯入備份
```

## v7 與 v6 差異

| 項目 | v6 | v7 |
|---|---|---|
| 任務管理 | 有 | 有 |
| 每日整理 | 有 | 有 |
| 本機通知 | 基礎提醒 | 可請求通知權限 |
| 任務提醒時間 | 無 | 有 |
| 行事曆匯出 | 無 | `.ics` 匯出 |
| 大型模型 | 有 | 有 |

## 資料庫

v7 使用新的 IndexedDB：

```text
MiniAI_PWA_v7
```

所以不會自動讀取 v6。請用匯出 / 匯入搬資料。
